## 2. MNIST CNN

    MNIST Dataset의 분류 작업을 시행하는 간단한 3-Layer CNN Model에 대한 Verilog 구현입니다.
    원 Model은 Pytorch로 구현하였으며, MNIST Dataset의 Testset에 대해 94%의 Accuracy를 갖습니다.
